LiquidView

A liquid media viewer for macOS. Drop an image, comic archive, video or document onto a screen edge — it pours out like water. Stick it to the edge, drag it off to float, or press F for full screen.

Download GitHub

Install — one line

curl -fsSL https://raw.githubusercontent.com/XPro-Gamer-Rhine/LiquidView-app/main/install.sh | bash

macOS 14+. Nothing else to install — mpv, ffmpeg and every library ship inside the app. The installer downloads with curl (no quarantine), copies to /Applications, and launches.

Opens practically anything

Images & photosJPEG, PNG, HEIC/HEIF, WebP, AVIF, JXL, TIFF, BMP, ICO, PSD, animated GIF, and camera RAW (CR2/CR3, NEF, ARW, DNG, RAF…). Zoom, pan, and page through the whole folder.
Video — every formatMP4, MOV, MKV, WebM, AVI, WMV, FLV, MPEG, .ts / .m2ts, VOB, 3GP, OGV, RMVB, MXF and more. mpv + VideoToolbox hardware decoding, full quality.
AudioMP3, FLAC, ALAC, AAC, M4A, WAV, AIFF, OGG/Opus, WMA, APE, DSF and more — with a scrubber, volume, and one-key mute.
Comic archivesRead pages straight out of a .zip / .cbz — no unzipping to disk. Numeric page order.
DocumentsPDF, CSV, TSV, XLSX, DOCX, Pages, Keynote, RTF, code and plain text — via the macOS QuickLook engine. Spreadsheets open wide; documents open like a page.
SubtitlesSidecar .srt / .ass / .vtt next to a movie load automatically.

Make it yours

ThemesTen built-in looks — colour palettes (Ocean, Forest, Sakura, Midnight, Mono…) and Liquid Glass / frosted materials that let the desktop show through. Pick from the menu bar; applies instantly.
Free resizeDrag the corner grip on a floating card to size it exactly how you like — media keeps its aspect, never distorts.
Any screen edgePour the viewer from the top, bottom, left or right — one edge or all four, across every display.

Three modes · one keyboard

Attached to an edge · Floating (drag it anywhere) · Full screen. The space around an attached card stays click-through, so the rest of your desktop keeps working.
